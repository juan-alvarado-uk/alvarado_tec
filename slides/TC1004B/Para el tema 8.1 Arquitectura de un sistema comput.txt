# Arquitectura de un sistema de cómputo

## ¿Qué es la arquitectura de un sistema de cómputo?

- Es la forma de seleccionar e interconectar componentes de hardware para crear computadoras según los requerimientos de funcionalidad, rendimiento y costo.

- Los componentes principales son: la placa principal o tarjeta madre, el microprocesador central o unidad central de proceso (CPU), el disco duro, el bus, la memoria principal y otros componentes controladores.

- La fuente de alimentación eléctrica proporciona la energía necesaria para el funcionamiento del sistema.
